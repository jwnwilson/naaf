# A2 Board + Issues List — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the primary Projects screen — the Board (Screen B: live-agents ribbon + 5 kanban columns) and the Issues List (Screen A: grouped rows) — composing the design system + data layer, toggled by the `?view` param.

**Architecture:** A `useProjectWorkItems(projectId)` hook fetches the project's flat work-item list (`/work-items?project=`); a pure `groupByStatus` util buckets items into the five status columns. `KanbanCard`/`ListRow` are composed presentational components (deferred here from Plan 1 because they consume the `WorkItem` type). `BoardScreen` renders `BoardView` or `ListView` per `?view`, with the `LiveAgentsRibbon` above the board. All wired through the existing AppShell/Topbar.

**Tech Stack:** Plan-1 `components/ui`, Plan-2 `lib/api` hooks + types, react-router `useParams`/`useSearchParams`.

`groupByStatus.ts`:
```ts
import type { components } from "../../lib/api/schema";

export type WorkItem = components["schemas"]["WorkItem"];
export type WorkItemStatus = WorkItem["status"];

export const STATUS_ORDER: WorkItemStatus[] = ["backlog", "todo", "in_progress", "in_review", "done"];

export function groupByStatus(items: WorkItem[]): Record<WorkItemStatus, WorkItem[]> {
  const out = Object.fromEntries(STATUS_ORDER.map((s) => [s, [] as WorkItem[]])) as Record<WorkItemStatus, WorkItem[]>;
  for (const it of items) (out[it.status] ??= []).push(it);
  return out;
}
```

import { useQuery } from "@tanstack/react-query";
import { apiList } from "../../lib/api/client";
import type { components } from "../../lib/api/schema";

export type WorkItem = components["schemas"]["WorkItem"];

export function useProjectWorkItems(projectId: string) {
  return useQuery({
    queryKey: ["work-items", "project", projectId],
    queryFn: () => apiList<WorkItem>("/work-items", { project: projectId }),
    enabled: Boolean(projectId),
  });
}
